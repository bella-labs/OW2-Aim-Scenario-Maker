## Bot Settings
### Return to Main Menu
**Description**: Returns to the Main Menu.

### Bot Slot
**Description**: The current bot to modify data for.

**Characteristics**

| Subitem  | Characteristics          | Value             |
| ---------| ------------------------ | ----------------- |
| 1        | Type                     | Integer           |
| 1        | Minimum                  | 0                 |
| 1        | Maximum                  | 11                |

**Menu Inputs**

| Input                    | Effect             |
| ------------------------ | ------------------ |
| Small Step Left          | -1                 |
| Small Step Right         | +1                 |
| Large Step Left          | -1                 |
| Large Step Right         | +1                 |
| Cycle Subitem            | N/A                |
| Assign Vector            | N/A                |
